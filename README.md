# S_G_H
> My Hospital Management System, developed using **C#** and powered by a **MySQL** database, is a practical solution designed to simplify healthcare administration and enhance patient care. With a focus on efficiency, accuracy, and data-driven decision-making, this project aims to simplify the complex processes that run within a hospital, ultimately improving the overall healthcare experience for both patients and medical staff.
